第二章：各类文件说明

2.1 规章制度（policies/）

正式发布的制度文件，具有公司内部效力。所有成员都应熟悉。

详细内容参考：policies/netdisk-usage-guide.md（公共网盘使用指南）

2.2 岗位要求（job-requirements/）

各岗位的详细职责描述，包括工作范围、协作关系、技能要求。

每个成员应了解自己岗位的要求，并可通过此目录了解其他成员的职责。

2.3 会议记录（meetings/）

公司重要会议的完整纪要，按日期归档。公开的会议记录会同步到官网。

2.4 通知存档（notifications/）

正式对外发布通知的存档。

2.5 自查活动（self-inspections/）

自查活动的执行记录和整改报告。

2.6 协作流程（collaboration/）

团队内部协作规范和流程文档，包括会议协作指南和 sub-agent 使用规范。

第三章：贡献指南

3.1 添加会议记录

在 meetings/YYYY-MM-DD/ 下创建 MD 文件，文件名即标题。

3.2 更新政策文件

在 policies/ 下编辑对应的 MD 文件。重大制度变更需经员工代表大会审议。

3.3 添加通知

在 notifications/ 下创建 YYYY-MM-DD-标题.md 文件。

3.4 官网内容更新

在 website/source/ 下编辑对应的 MD 文件，然后运行编译脚本发布。

3.5 Git 使用规范

cd ~/NetDisk
git add <files>
git commit -m "类型：简要描述"
git push origin master

第四章：禁止事项

  • 禁止在 NetDisk 中存储密码、密钥、token 等敏感凭证
  • 禁止提交空文件或无意义的占位文件
  • 禁止在未经审议的情况下修改 policies/ 下的正式制度文件
